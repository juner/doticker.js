doticker.js

Share JavaScript, HTML5 and CSS

何するモノ？

doticker.js は、Twitter 公式のプロフィールウィジェット のように、jsdo.it で公開したコードをブログに流してくれる jQuery プラグインです。

使い方

<div id="ticker"></div>

<script type="text/javascript" src="jquery.min.js"></script>
<script type="text/javascript" src="doticker.min.js"></script>
<script>
$(function() {
	$("#ticker").doticker({
		// 動作に関するオプション
		name: "tokkonopapa",
		loop: true,
		maxCodes: 10,
		interval: 6000,
		scrollbar: false,
		duration: "slow",
		// 見た目に関するオプション
		width: "250px",
		height: "300px",
		background: "#9bc6f2",
		bodyground: "#fff",
		bordercolor: "#888"
	});
});
</script>

各パラメータの詳細は、README.md をご覧ください。
